# Testing Asciinema Embedding

This is a test post to demonstrate asciinema embedding functionality.

## Basic Asciinema Embed

Here's a simple asciinema recording:

[asciinema:demo.cast]

## Asciinema with Options

Here's the same recording with some options:

[asciinema:demo.cast autoplay=true theme=monokai speed=1.5 caption="A demo terminal session"]

## Another Example

You can add multiple recordings in a single post:

[asciinema:demo.cast loop=true speed=0.8 fit=width caption="Looped demo at slower speed"]

That's it! The asciinema player should now work in your blog posts.
